
# 👩‍💻 majority-checker

Ce programme en **C** détermine si chaque personne est **majeure** ou **mineure** selon son **âge** et son **sexe**. 🎂👨‍🦱👧  
Il affiche ensuite le total de femmes et d’hommes majeurs et mineurs. 📊🧮

## 📝 Prérequis
Avant d’exécuter le programme, assurez-vous d’avoir :  
- 💻 Un ordinateur avec un système compatible (Linux, macOS, Windows)  
- 🛠️ **GCC** ou un autre compilateur C installé  
- 📂 Le fichier source `main.c` du projet  
- 🖊️ Accès pour saisir les données au clavier (année de naissance, sexe)  

## ⚙️ Compilation et exécution

```bash
  # Compilation
  gcc main.c -o main

  # Exécution
  ./main
```
